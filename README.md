# Pong Game

Teaching myself MongoGame, as my first project I decided to create a classic game of Pong using MonoGame.

## Features

- Two-player Side by Side
- Score tracking (coming soon)

## Getting Started

### Prerequisites

- .NET SDK
- MonoGame

## Controls

- **Player 1**:
    - Move Up: `W`
    - Move Down: `S`
- **Player 2**:
    - Move Up: `Arrow Up`
    - Move Down: `Arrow Down`
